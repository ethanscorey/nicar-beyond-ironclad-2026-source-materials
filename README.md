# NICAR Beyond Ironclad 2026

Beginner-friendly training materials for a hands-on fact-checking workshop in data journalism.

## Notebook
- `BeyondIronclad.ipynb`
Note that this notebook was designed to [run in Google Colab](https://colab.research.google.com/drive/1NrbfS4ik_przKU74RyuWhBvcFFTQVQmN), allowing participants to run it on their own devices without setting up Python environments. To run it locally, I recommend removing the `!git clone ...` and `%pip install ...` code blocks at the start of the notebook.

## Dummy datasets
- `data/mass_transit_delays.csv`
- `data/county_population.csv`
- `data/county_air_pollution.csv`
- `data/income_demographics.csv`
- `data/race_ethnicity_versions.csv`

Each dataset has a companion data dictionary in the same folder (`*_dictionary.csv`).
